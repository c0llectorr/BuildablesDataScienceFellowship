Required Columns in CSV:

Your CSV file must contain the following columns:

  • sy_snum - Number of stars in the system
  • sy_pnum - Number of planets in the system
  • disc_year - Year of discovery
  • pl_orbper - Orbital period of the planet (days)
  • pl_orbsmax - Semi-major axis of the planet's orbit (AU)
  • pl_rade - Planetary radius (Earth radii)
  • pl_radj - Planetary radius (Jupiter radii)
  • pl_bmasse - Planetary mass (Earth masses)
  • pl_bmassj - Planetary mass (Jupiter masses)
  • pl_orbeccen - Orbital eccentricity
  • pl_insol - Insolation flux (Earth flux)
  • pl_eqt - Planet equilibrium temperature (Kelvin)
  • st_teff - Effective temperature of the host star (Kelvin)
  • st_rad - Stellar radius (Solar radii)
  • st_mass - Stellar mass (Solar masses)
  • st_met - Stellar metallicity
  • st_logg - Stellar surface gravity
  • sy_dist - Distance to the system (parsecs)
  • discoverymethod (optional) - Method used for detection

Note: If the CSV contains a 'disposition' column with values 'CONFIRMED' or 'CANDIDATE', the application will calculate evaluation metrics.
