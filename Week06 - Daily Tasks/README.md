# **Book-Recommendation-System**
A book recommendation system Notebook that creates, evaluates, and scores the recommender system developed. 
It is a practice task to understand how recommender systems are built.

### **All work is done on Microsoft Fabric in LakeHouse.**
The LakeHouse includes the following files:
1. BookRecommendationModel
2. Dataset
3. Notebook

Due to Microsoft's policy, we could not add the datasets and the model to the repository.
